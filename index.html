PHP基础认识

发表于 2016-11-10

php语法

格式为.php结尾

内容写在<?php ?>中间

echo 为输出方法

<?php echo "script"; ?>

  • php页面转utf-8编码

header("Content-Type: text/html;charset=utf-8");

  • 有三种注释方法

  • //单行注释

  • # 单行注释

  • /多行注释 /

变量的声明

  • 变量以\$开头
  • 变量名由字母或者下划线开头,由字母,下划线,数字组成
  • 区分大小写
  • php和js一样是弱类型语言,不需要事先声明变量的数据类型
  • 例子:
    $varible

赋值

  • 引用赋值

  • 变量的变量

  • PHP的超全局变量

    • $_SERVER 服务器变量,包含着服务器和客户端配置及当前请求环境的有关信息
    • $_SERVER['SERVER_NAME']服务器主机名
    • $_SERVER['REMOTE_ADDR']客户端IP地址
    • $_SERVER['REQUEST_URI']URL的路径部分
    • $_SERVER['HTTP_USER_AGENT']操作系统和浏览器的有关信息

  • 常量

    • 常量指的是在程序执行中无法修改的值,如PI
    • 在脚本执行期间该值不能改变
    • 对大小写敏感,常量名通常是大写的
    • 常量是全局的,可以在脚本的任何地方引用
    • 分为内置常量和自定义常量
    • 常量用define()函数定义

    • 常量的定义

    • 内置常量

      • PHP_OSphp所在的操作系统名字
      • PHP_VERSION当前php版本
    • 魔术常量
      • __LINE__文件中的当前行号
      • __FILE__文件的完整路径和文件名
      • __FUNCTION__函数的名称
      • __CLASS__类的名称
      • __METHOD__类的方法名

算术运算符

  • + - * / %同JS一样
  • "ssdf" + 1;中字符串会被转化为0
  • "12sf" + 1;中字符串会被转化为12

赋值运算符

  • = += -= *= /=

字符串运算符

  • .=
  • $a .= 5拼接赋值相当于$a = $a.5

递增和递减运算符

  • ++$i 先加,$i的值加1,然后再返回$i的值
  • $i++ 后加,先返回$i的值,再加
  • --$i 同理
  • $i-- 同理

逻辑运算符

  • && AND 逻辑与
  • || OR 逻辑或
  • ! NOT 逻辑非

比较运算符

  • > < >= <=
  • != <> 不等于
  • == === !==同JS一样
